# Rusalka

Welcome to **Rusalka** – a repository for my personal portfolio, blog, and services website! Named after the Slavic water spirits, which lured heros to the waterworld with the intent to release them centuries later (which is exactly what I'm tryna do as a result of getting clients and keeping them happy for a long time!)

---

## What's This All About?

This is my playground for learning modern web development while building something actually useful. The stack is pretty straightforward:

- **Frontend**: `Next.js` - because apparently everyone's doing it, and I figured I should learn it for survival reasons, might change later to something more lightweight, we'll see
- **Backend**: `Go` - perfect excuse to learn Go, plus it's become the `Go`-to (sorry, I had to do it, I could not help myself) language for backends

I'll probably add more technologies in here as I go.

The purpose of this is to build and learn the basic functionality that I can use in future projects, whether they are commercially driven or passion projects, and to share them with the world.

## Why Open Source?

Why keep it locked away? The code's out here for anyone curious enough to peek under the hood. Plus, I'll probably end up recycling bits and pieces for other projects down the road.

## Current Status

This is very much a work in progress as I figure out all the necessary technologies. Expect some experimental code, the occasional "why did I do it this way?" moment, and probably a few rewrites along the way.
